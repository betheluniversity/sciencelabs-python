NEW FALL 2021!

Sessions no longer need a virtual option. So Zoom Urls are now optional for any type of session.

This page is used to add zoom urls to sessions in a schedule so that when a student joins a virtual help session, they have access to the session zoom url.

To add a zoom url to a schedule, simply type or paste the zoom url into the zoom url section and hit the Save Zoom Urls button to save the url. You can either do all urls at once and then hit save or do them one at a time.

To delete a zoom url, simply delete the zoom url for that course and hit Save Zoom Urls.

To change a zoom url, simply paste a new zoom url where the previous one was and hit Save Zoom Urls.
