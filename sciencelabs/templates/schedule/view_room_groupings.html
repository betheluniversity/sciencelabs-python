This page is used to edit room group capacities.

Room group capacities are automatically set to the total capacity of all sessions in the room group, but you may want to change this since it may not be accurately depicting how much capacity the rooms can handle.

A room group is defined as all the sessions that have the exact same date, start time, end time, room number (including the same capitalization).

Adding sessions to a room group will allow students from different sessions all to sign into their respective sessions using one login page on one laptop. This is useful when you want to have multiple sessions in one room, but only want to use one laptop to sign people in.

To edit room group capacity simply input a number next to the room group you wish to change the capacity and hit the Save Room Group Capacity button. checkbox and hit the Save button.

The total capacity of all sessions in a room group should be less than or equal to the capacity of the room group.
