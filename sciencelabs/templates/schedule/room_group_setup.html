This page is used to add all sessions in a schedule to a room grouping.

A room group is defined as all the sessions that have the exact same date, start time, end time, and room number.

Adding sessions to a room group will allow students from different sessions all to sign into the same room using one login page.

To add all sessions in a schedule to a room grouping, simply check the "Add to Room Group" checkbox and hit the Save button.

To delete all sessions in a schedule to a room grouping, simply uncheck the "Add to Room Group" checkbox and hit the Save button.
