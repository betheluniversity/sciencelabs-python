{{ app_settings['LAB_TITLE'] }} Courses for ***SEMESTER YEAR***

This page has two sections, the "Math Lab Courses Across All Terms" and the "Math Lab Courses for Current Term". Administrators should populate the bottom "Math Lab Courses Across All Terms" section with all courses that this lab serves across all terms.

To add courses to the "Math Lab Courses Across All Terms" section, click on the "add a course" text in the box, type a course number (ex. MAT124M, use appropriate capitalization and no spaces) and press enter. When you have added all applicable courses, press Submit Courses.

Each new course number will now be checked against Banner; if the course number is valid, the course name will be added to the course number in the list and all sections of the course will populate the "Math Lab Courses for Current Term" list at the top of the page. If the course number is not valid, an error message will appear at the top of the screen.

Math Lab Courses Across All Terms determines which courses will be available to be offered at help sessions. If you no longer want a course to be offered at future help sessions, you should delete it from the "Math Lab Courses Across All Terms" section. In the bottom Math Lab Course list, press the "x" next to the course name and number. When you have removed all applicable courses, press Submit Courses. This will NOT remove the course from the current term course listing found at the top of this page.

Math Lab Courses for Current Term determines which courses will be available for reports. If you no longer want a course to be available for reports, you should delete it from the "Math Lab Courses for Current Term" section. To delete a course in this section, click on the course title, and then press Delete in the following window to delete the course. Make sure this is done for all sections of the course, if desired. Please note that courses from past terms will still be available for reporting. However, all courses that are not offered in the current term will NOT appear in the course list.

Further course information is available for each course by clicking on the course name. Information including the instructor, room, day and time of the course along with the number enrolled, is pulled from Banner and displayed.
