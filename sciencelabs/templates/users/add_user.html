Search for a user to add by entering first name and last name (enter name or first few characters). Once you have entered a first and last name, press enter or the Search button. When the user appears, press AddUser. If the user does not appear, check the spelling of the user name as the system is pulling user data from Banner.

* Professors are automatically added to the user list when their courses are pulled from Banner when the current term is selected, and need not be added manually.

* Users with first or last names which include spaces may need to be searched by only the first part of their first or last name.
